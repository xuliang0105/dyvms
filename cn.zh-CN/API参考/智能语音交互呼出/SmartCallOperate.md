# SmartCallOperate {#doc_api_Dyvmsapi_SmartCallOperate .reference}

调用接口SmartCallOperate在智能外呼通话中，发起指定动作。

在智能语音交互呼出通话中，转接呼叫中心坐席时，可以使用接口**SmartCallOperate**指定被叫号码发起指定动作。

**说明：** 当前仅支持指定动作为被叫号码与呼叫中心坐席桥接。
